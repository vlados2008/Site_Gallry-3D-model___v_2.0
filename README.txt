  2. ОПИСАНИЕ КАЖДОГО ФАЙЛА
================================================================================

  index.html
  ----------
  Единственная HTML страница. Содержит весь контент:
  - Preloader (экран загрузки)
  - Header с навигацией
  - Hero секция с заголовком и статистикой
  - Gallery с карточками и фильтрами
  - About секция
  - Contact / Telegram секция
  - Footer
  - Модальное окно для деталей модели
  - Кнопка "Наверх"

  css/variables.css
  -----------------
  Здесь хранятся ВСЕ дизайн-токены:
  - --accent: основной акцентный цвет (голубой)
  - --accent-2: вторичный акцент (фиолетовый)
  - Цвета фонов, текста, границ
  - Тёмная и светлая темы (меняются через data-theme="dark/light")
  - Шрифты, размеры, отступы, z-index
  → Чтобы перекрасить весь сайт — меняй только этот файл!

  css/components.css
  ------------------
  Переиспользуемые UI компоненты:
  - .btn, .btn--primary, .btn--ghost, .btn--telegram
  - .model-card (карточки галереи)
  - .filter__btn (кнопки фильтрации)
  - .theme-toggle (кнопка темы)
  - .scroll-top (кнопка наверх)
  - .modal (модальное окно)
  - .skill-chip, .contact-chip

  css/layout.css
  --------------
  Структура каждой секции страницы:
  - .preloader (экран загрузки)
  - .header, .nav (шапка)
  - .mobile-menu (мобильное меню)
  - .hero (главный экран)
  - .gallery, .gallery__grid (сетка карточек)
  - .about__container (секция обо мне)
  - .telegram-block (Telegram контакт)
  - .footer (подвал)

  js/data.js
  ----------
  ★ САМЫЙ ВАЖНЫЙ ФАЙЛ ДЛЯ ДОБАВЛЕНИЯ МОДЕЛЕЙ ★
  Массив MODELS_DATA со всеми моделями галереи.
  Каждый объект содержит:
    id, title, category, categoryLabel,
    description, tags, placeholderClass, svgIcon
  → Добавь сюда новую модель — она появится в галерее автоматически!

  js/gallery.js
  -------------
  Фильтрация карточек по категориям. При клике на кнопку фильтра
  карточки не подходящей категории скрываются с анимацией.

  js/modal.js
  -----------
  Модальное окно. При клике на карточку или "View Details"
  открывается попап с расширенным описанием модели.
  Закрывается по: клику на фон, кнопке X, нажатию ESC.

  js/theme.js
  -----------
  Переключение тёмной/светлой темы. Сохраняет выбор в localStorage.
  При следующем визите тема восстанавливается автоматически.

  js/particles.js
  ---------------
  Canvas-анимация частиц на hero секции. Лёгкая, не нагружает GPU.
  Автоматически отключается если пользователь включил
  "Reduce motion" в системных настройках доступности.

  js/nav.js
  ---------
  - Тень на шапке при скролле
  - Подсветка активного пункта меню
  - Мобильное меню (hamburger)
  - Плавный скролл к секциям
  - Кнопка "Наверх" (появляется после 400px скролла)

  js/animations.js
  ----------------
  IntersectionObserver следит за элементами с классами
  .reveal-up и .reveal-fade. Когда элемент входит в экран —
  добавляет класс .is-visible который запускает CSS анимацию.

  js/counter.js
  -------------
  Анимированные числа (40+, 3+, 6+) в hero секции.
  Число "накручивается" при первом попадании в видимость.

  js/preloader.js
  ---------------
  Прогресс-бар при загрузке. Скрывает preloader когда страница
  полностью загружена (событие 'load'). Максимум 4 секунды.

  js/main.js
  ----------
  Точка сборки: устанавливает текущий год в footer.

  3. КАК ЗАПУСТИТЬ САЙТ
================================================================================

  ВАРИАНТ 1 — Простой (рекомендуется для начала):
  ------------------------------------------------
  Просто открой файл index.html в браузере (Chrome / Firefox / Edge).
  Двойной клик по файлу или перетащи в браузер.

  ⚠ ВНИМАНИЕ: Google Fonts и некоторые эффекты требуют интернета.
     Без интернета шрифты заменятся системными (монospace/sans-serif).

  ВАРИАНТ 2 — Через локальный сервер (правильный способ):
  --------------------------------------------------------
  Установи VS Code и расширение "Live Server" (Ritwick Dey).
  1. Открой папку portfolio/ в VS Code
  2. Правый клик на index.html → "Open with Live Server"
  3. Сайт откроется на http://localhost:5500

  Или через Python (если установлен):
    cd portfolio/
    python3 -m http.server 8080
  Затем открой http://localhost:8080

  ВАРИАНТ 3 — Python Flask backend:
  ----------------------------------
  Если захочешь добавить backend (форма связи, динамику):
    cd portfolio/python/
    pip install flask
    python3 app.py
  (файл app.py нужно создать самостоятельно или попросить ChatGPT)

  4. КАК ДОБАВИТЬ СВОИ ИЗОБРАЖЕНИЯ (ЗАМЕНИТЬ ЗАГЛУШКИ)
================================================================================

  Шаг 1: Сделай рендер своей модели в Blender.
          Рекомендуемые размеры: 1600×1200px или 1920×1080px
          Форматы: JPG (меньше вес) или PNG (лучше качество)

  Шаг 2: Назови файл в том же формате что и заглушки:
          model-01-cyber-warrior.jpg
          model-02-neon-cruiser.jpg
          и т.д.

  Шаг 3: Положи файлы в папку:
          portfolio/assets/images/

  Шаг 4: В index.html найди тег <img> для нужной карточки и замени:
          src="assets/images/model-01-cyber-warrior.svg"
          на:
          src="assets/images/model-01-cyber-warrior.jpg"

  ✓ Заглушка автоматически скроется если img загрузится успешно.


================================================================================
  5. КАК ДОБАВИТЬ НОВУЮ МОДЕЛЬ В ГАЛЕРЕЮ
================================================================================

  Открой файл js/data.js и добавь новый объект в массив MODELS_DATA:

  {
    id: 10,                              // Уникальный ID (следующий по порядку)
    title: 'Название модели',
    category: 'scifi',                   // Одно из: characters | vehicles |
                                         //  weapons | buildings | scifi | other
    categoryLabel: 'Sci-Fi',            // Отображаемое название
    description: 'Описание модели...',
    tags: ['PBR', 'High-Poly'],          // До 3 тегов
    placeholderClass: 'model-card__placeholder--scifi',
    svgIcon: `<svg>...</svg>`,           // SVG иконка для заглушки
  },

  Затем в index.html добавь карточку по образцу существующих карточек
  (скопируй любую article.model-card и измени данные + data-id).

  Или: в будущем можно сделать рендер карточек из data.js автоматически
  через JS — это несложно добавить в gallery.js.

================================================================================
  8. КАК ПОМЕНЯТЬ ЦВЕТА И ТЕМУ
================================================================================

  Все цвета находятся в css/variables.css

  Основные переменные:
    --accent:   #5DE8FF   ← Голубой акцент (можно поменять на другой цвет)
    --accent-2: #A78BFA   ← Фиолетовый акцент

  Тёмная тема: блок [data-theme="dark"]
  Светлая тема: блок [data-theme="light"]

  По умолчанию сайт открывается в тёмной теме.
  Чтобы поменять — открой js/theme.js и измени:
    const DARK = 'dark';  →  default на light

  Шрифты (в :root):
    --font-display: 'Orbitron'      ← Заголовки (cyberpunk стиль)
    --font-body:    'Syne'          ← Основной текст
    --font-mono:    'JetBrains Mono' ← Метки, теги, код
